# WVU Cloud Deployment Docs

## Purpose
This repository provides clear documentation for deploying course projects to the cloud using the WVU workflow.

## What This Is For
This site is intended for WVU professors, instructors, and staff who need a step-by-step reference for project setup, GitHub configuration, Azure deployment, and troubleshooting regarding the CS330 class.
